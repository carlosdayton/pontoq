# 🍣 Ponto Q — Landing Page

> Landing page profissional para o **Ponto Q**, lanchonete, padaria e sushi localizado em Luziânia-GO.

---

## ✨ Destaques

- **Animações cinematográficas** com [GSAP](https://gsap.com/) + ScrollTrigger
- **Parallax** no hero e na seção CTA
- **Cursor customizado** com efeito magnético
- Contadores animados, stagger de elementos e reveal no scroll
- Design **dark premium** com paleta dourada
- 100% **responsivo** (mobile, tablet, desktop)
- Zero dependências de build — abre direto no navegador

---

## 🗂 Estrutura

```
ponto-q/
├── index.html          # Estrutura da página
├── style.css           # Estilos e estados iniciais das animações
├── gsap-animations.js  # Todas as animações GSAP
└── img/                # Fotos do local
```

---

## 📄 Seções

| Seção | Descrição |
|---|---|
| **Hero** | Fullscreen com parallax e entrada cinematográfica |
| **Sobre** | História do restaurante com fotos sobrepostas e contador animado |
| **Diferenciais** | 4 cards com hover e texto decorativo parallax |
| **Cardápio** | Abas (Sushi · Lanches · Tapiocas · Bebidas) com preços reais |
| **Instagram** | CTA direto para `@quitandasjt` |
| **Depoimentos** | 3 avaliações do Google Maps |
| **CTA** | Botões para WhatsApp e pedido pelo site |
| **Contato** | Endereço, horários, redes sociais e mapa embarcado |

---

## 🚀 Como rodar

Sem build, sem dependências. Basta abrir:

```bash
# Clone o repositório
git clone https://github.com/SEU_USUARIO/ponto-q.git

# Abra no navegador
open ponto-q/index.html
```

> As fontes (Google Fonts) e o GSAP são carregados via CDN — necessário conexão com a internet.
